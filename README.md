# Shubham Raj - AI Engineer Portfolio

A professional portfolio website for Shubham Raj, a Senior AI Engineer specializing in LLMOps, NLP, and Document AI.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (comes with Node.js)

### Installation & Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The portfolio will be available at `http://localhost:5173` (or the port shown in your terminal).

## Making Changes

After making changes to the code:

1. Add your changes:
```bash
git add .
```

2. Commit with a message:
```bash
git commit -m "Your message here"
```

3. Push to the repository:
```bash
git push origin main
```

## Deployment

To build and deploy the portfolio:

```bash
npm run build
npm run deploy
```

The portfolio is deployed on GitHub Pages (gh-pages branch).

## Portfolio Sections

- **Hero**: Introduction and personal branding
- **About**: Professional summary and key expertise areas
- **Experience**: Work history and education timeline
- **Skills**: Technical skills organized by category
- **Projects**: Showcase of notable AI/ML projects
- **Contact**: Contact information and social links

## Technologies Used

- React.js
- Three.js (3D graphics)
- Tailwind CSS
- Framer Motion (animations)
- Vite (build tool)
